Your Website

This is an example of how to embed the TalkDocu Chatbot into your existing website.

Simply add the script tag and initialize the chatbot with your conversation ID and auth token.

Embed Code

<!-- Add this script to the end of your body tag -->
<script src="talkdocu-chatbot.js"></script>
<script>
  // Initialize the chatbot
  document.addEventListener('DOMContentLoaded', function() {
    const chatbot = new TalkDocuChatbot({
      convStringId: 'YOUR_CONVERSATION_ID', // Replace with your actual conversation ID
      authToken: 'YOUR_AUTH_TOKEN',        // Replace with your actual auth token
      apiBase: '/api'                       // Change if your API is hosted elsewhere
    });
  });
</script>
