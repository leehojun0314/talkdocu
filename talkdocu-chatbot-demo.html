How to Use

To use the TalkDocu Chatbot on your website, follow these steps:

1. Include the script

<script src="talkdocu-chatbot.js"></script>

2. Initialize the chatbot

// Option 1: Initialize when creating the chatbot
const chatbot = new TalkDocuChatbot({
  convStringId: 'your-conversation-id',
  authToken: 'your-auth-token'
});

// Option 2: Initialize later
const chatbot = new TalkDocuChatbot();
chatbot.setConversation('your-conversation-id', 'your-auth-token');

3. Customize (Optional)

You can customize the chatbot by passing additional options to the constructor: 

const chatbot = new TalkDocuChatbot({
  convStringId: 'your-conversation-id',
  authToken: 'your-auth-token',
  apiBase: '/api', // Change if your API is hosted elsewhere
  salutation: 'Hello! How can I help you today?'
});
